# GIS & Remote Sensing Journals — Complete Directory

A comprehensive directory of journals relevant to GIS, remote sensing, and geospatial science researchers. Covers top-tier journals ranked by impact factor, open access options, GIS-adjacent venues, Chinese and regional journals, conference proceedings, preprint servers, and strategic journal selection guidance.

> **Quick Picks**
> - **Highest impact (RS):** Remote Sensing of Environment (IF ~13.5), ISPRS J-PARS (IF ~12.7)
> - **GIScience theory:** IJGIS (IF ~3.5) — the flagship GIScience journal
> - **Applied + Open Access:** JAG/Int. J. Applied Earth Obs. (IF ~7.5, Gold OA)
> - **Software papers:** JOSS (free, peer-reviewed, DOI for your code)
> - **Fast review:** Computers & Geosciences (2-4 months typical)
> - **Free OA:** Geo-spatial Information Science, JOSS, ISPRS Archives

### Conference Paper vs. Journal Paper

| Aspect | Conference Paper | Journal Paper |
|--------|-----------------|---------------|
| Length | 4-10 pages | 15-40 pages |
| Review rounds | 1 (typically) | 2-3 |
| Review time | 2-3 months | 3-12 months |
| Revision allowed | Rarely | Yes |
| Presentation | Required | Optional |
| Prestige (GIS) | Medium | High |
| Citable | Yes | Yes |
| Strategy | Present early work, get feedback, network | Full study, establish contribution |

### Conference Selection Strategy

```
Is your work complete with full results?
  YES → Submit to a journal (stronger contribution record)
  NO  → Is it a promising early-stage idea?
          YES → Submit to a workshop (SIGSPATIAL workshops, AGILE)
          NO  → Is it a mid-stage project with preliminary results?
                  YES → Submit to a conference (AGILE, GIScience, SIGSPATIAL)
                  NO  → Not ready — continue research
```

### Preprint-to-Journal Workflow

```
1. Finish manuscript draft
   │
2. Submit to preprint server (EarthArXiv recommended for GIS/RS)
   │ → Receive DOI within 1-2 days
   │ → Share for early feedback
   │
3. Submit to journal (most GIS journals accept papers with preprints)
   │ → Journal reviews proceed normally
   │
4. After acceptance:
   │ → Update preprint with "Published in [Journal]" link
   │ → Check Sherpa Romeo for publisher policy
   │
5. Post-publication:
   → Preprint remains as green OA version
   → Journal version is the version of record
```

## Journal Selection Decision Trees

### By Paper Type

```
Is your paper primarily about a NEW METHOD/ALGORITHM?
  YES → Is it deep learning / ML focused?
          YES → TGRS, JSTARS, RSE, or ISPRS J-PARS
          NO  → IJGIS, C&G, or Geoinformatica
  NO  → Is it an APPLICATION paper?
          YES → Does it use remote sensing data?
                  YES → RSE, ISPRS J-PARS, JAG, or JSTARS
                  NO  → IJGIS, Transactions in GIS, or domain journal
          NO  → Is it a REVIEW/SURVEY paper?
                  YES → Earth-Science Reviews, IJGIS, RSE, or ISPRS
                  NO  → Is it a SOFTWARE/TOOL paper?
                          YES → JOSS, C&G, or JORS
                          NO  → Is it a DATA DESCRIPTOR?
                                  YES → ESSD, Scientific Data, or Data in Brief
                                  NO  → Consider target audience and scope
```

### By Career Stage

```
PhD Student (first paper):
  → Mid-tier journals: C&G, JAG, Transactions in GIS
  → Rationale: Realistic acceptance, reasonable review time, good visibility

PhD Student (2nd+ paper):
  → High-tier: IJGIS, TGRS, JSTARS
  → If strong results: RSE, ISPRS J-PARS

Postdoc (career building):
  → Top-tier: RSE, ISPRS J-PARS, Nature Geoscience (if appropriate)
  → Balance: Mix of top-tier and high-tier for productivity

Assistant Professor (tenure track):
  → Focus on top-tier journals for tenure case
  → Supplement with invited reviews and editorial roles
```

### By Open Access Requirement

```
Funder requires Gold OA?
  YES → Free OA: Geo-spatial Info Sci, JOSS, JORS
        Paid OA: JAG ($3,500), MDPI Remote Sensing (~$2,700)
        Check: Institutional Read & Publish agreements
  NO  → Hybrid journals: Submit as subscription, self-archive preprint
        Green OA: Post accepted manuscript after embargo
```

## Predatory Journal Warning Signs

### Red Flags

- Unsolicited email invitations to publish or join editorial boards
- Guaranteed acceptance or extremely fast review (days instead of months)
- No clear editorial board or editorial board members are unverifiable
- Missing or fake ISSN
- Not indexed in Scopus, Web of Science, or DOAJ
- APC not clearly stated upfront
- Journal name mimics a legitimate journal
- No clear peer review process described

### Verification Resources

| Resource | URL | Purpose |
|----------|-----|---------|
| DOAJ | https://doaj.org | Directory of legitimate OA journals |
| Beall's List (archived) | Various mirrors | Known predatory publishers |
| Think. Check. Submit. | https://thinkchecksubmit.org | Journal verification checklist |
| Compass to Publish | https://app.lib.uliege.be/compass | Journal evaluation tool |
| JCR / Scopus | Via institution | Indexed = legitimate |
| Sherpa Romeo | https://v2.sherpa.ac.uk/romeo/ | Publisher policies |

### Safe Submission Checklist

- [ ] Journal is indexed in Web of Science or Scopus
- [ ] Journal appears in DOAJ (if OA)
- [ ] Editorial board members are verifiable researchers
- [ ] You can find published articles in the journal that are cited by others
- [ ] APC and review process are clearly documented
- [ ] Journal has a real, functional peer review process
- [ ] You did NOT learn about the journal from an unsolicited email

## Emerging Trends in GIS Publishing (2025)

### Overlay Journals
Journals that review preprints rather than hosting manuscripts themselves. Lower cost, faster publication.
- **Discrete Analysis** (math, model for GIS overlay journals)
- **Peer Community In (PCI)** — Free overlay peer review for preprints

### Registered Reports
Submit your study design before collecting data. If accepted, the paper is published regardless of results.
- Eliminates publication bias
- Supported by some environmental/ecology journals

### Open Peer Review
Reviews are published alongside the paper.
- **Copernicus journals** (e.g., ESSD, GMD) — Open review by default
- **F1000Research** — Full open review with post-publication revisions

### Data Journals
Journals specifically for publishing datasets with peer review.
- **ESSD** (IF ~11.2) — Highest impact data journal
- **Scientific Data** (IF ~5.8) — Nature portfolio
- **Data in Brief** (Elsevier) — Brief data articles
- **Geoscience Data Journal** (Wiley/RMetS) — Geoscience datasets

### Micro-Publications
Short, focused papers on single findings or negative results.
- **JOSS** — Software papers (typically 1-2 pages of text)
- **Brief Communications** in various journals
- **Research Notes of the AAS** — Astronomy (model for GIS)
