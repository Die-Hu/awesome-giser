# Visualization

> Techniques and tools for effective geospatial visualization -- from static thematic maps to interactive 3D dashboards.

## Overview

Geospatial visualization transforms raw spatial data into meaningful visual representations. This module covers the full spectrum of approaches: traditional cartographic design, dynamic temporal animations, immersive 3D renderings, and interactive dashboard applications.

## Highlight

This module takes you from **static thematic maps** to **interactive 3D dashboards**:

1. **Thematic Maps** -- Learn classification methods and color schemes for effective data communication.
2. **Temporal Animation** -- Add the time dimension to reveal patterns and trends.
3. **3D Visualization** -- Create immersive views of terrain, buildings, and point clouds.
4. **Dashboards** -- Build interactive applications for data exploration and storytelling.
5. **Cartography & Design** -- Apply design principles that make maps clear, accessible, and beautiful.

## Gallery of Approaches

| Visualization Type | When To Use | Go-To Tool |
|--------------------|-------------|------------|
| Choropleth map | Show rates/ratios across regions | D3.js, QGIS, Kepler.gl |
| Heat map | Show point density | MapLibre heatmap, deck.gl |
| Flow map | Show origin-destination movement | Flowmap.blue, deck.gl ArcLayer |
| 3D terrain | Show topography | CesiumJS, MapLibre 3D |
| Point cloud | Show LiDAR/photogrammetry | Potree, CesiumJS |
| Time animation | Show change over time | Kepler.gl, deck.gl TripsLayer |
| Dashboard | Explore data interactively | Streamlit, Observable Framework |
